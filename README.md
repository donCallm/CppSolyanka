# CppSolyanka
First project on c++

---

___Installing dependencies___
```
sudo apt update
sudo apt install cmake
```

___Buil project___
```
cd CppSolyanka
mkdir build && cd build
cmake ..
make -j
```

___Server start___
```
cd CppSolyanka
cd build
cd src
cd server
./server
```

___Client start___
```
cd CppSolyanka
cd build
cd src
cd client
./client
```
